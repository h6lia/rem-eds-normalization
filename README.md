REM/EDS normalization without oxygen

Python tool for geochemical processing of REM/EDS data.

## What it does
- reads CSV and Excel files
- removes oxygen from elemental datasets
- normalizes remaining elemental concentrations to 100 wt%
- exports cleaned output tables
- can create a plot of normalized element distributions

## Requirements
- Python 3
- pandas
- matplotlib

Note: Developed with AI-assisted coding. Outputs should be critically reviewed before use in scientific interpretation.

```bash
python main.py
